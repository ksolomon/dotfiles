# dotfiles - My personal collection of dotfiles for Linux & Mac machines.

.bashrc/.zshrc will use extra files (dircolors, aliases, etc) from the repo.  You can add machine-specific aliases to ~/.local_aliases and .bashrc/.zshrc will pick these up.
